# bobo

KISS static site generator.

## Getting Started

```
$ git clone https://github.com/cristianrz/bobo.git
```

### Prerequisites

* `git`
* `markdown`/`discount`
* `busybox`

### How to use

```
$ ./bobo-init
```

Now edit your website and

```
$ ./bobo-run
```

and you have a local server at `localhost:8080`
